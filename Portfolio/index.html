Work

You can check out all my projects and research assignments here on GitHub!

Blitz-Build React App

Blitz-Build is a a full fledged React web-application that I worked on with a team of six full-time Lambda students. Blitz-Build is a real world application, made to assist residential construction teams in keeping track of their work and coordinating with their teams. For this project, my main focus was fulfilling the objectives of our stake holder, in particular through managing the state management and navigation for the application. You can check out Blitz-Build here!

Sleep-Tracker React App

For Sleep-Tracker, I worked on creating a functioning back-end using Node.js as well as the front-end functionality for the "Homepage" and "Login" portions of the wbsite. I ecspecially enjoyed creating a bar-graph represntation for the users inputed data as well as creating a fun, easy to navigate aesthetic for the site. You can check out my Sleep-Tracker web-app here!

Thunderball Run, a React App with a Django back-end

Thunderball Run is a maze based navigation game, which utilizes a React based front-end and a Django based back-end. For this project I had to combine my javascript skills with my python skills along with some creative css, to create something that felt really special! I learned about visiualizing data to create a map, utilizing multiple data structures (ecspecially python dictionaries and classes), as well as made big leaps with my css aesthetic skills! You can check out Thunder-ball Run here!
